Allocates seats based on party total votes, and assigns winning candidates based on their popularity within the party. 

Inherits from the abstract Class \mbox{\hyperlink{classElectionSystem}{Election\+System}}, and overrides the method. This function will find the quota to do the first seat allocation. It will then move on to the second seat allocation, where it will see if there are possible ties between the parties. Handling ties will be handled by another class, this function can get the result. After all seats are allocated to the parties, we then have to find the winning candidates within the parties, and handle any ties between candidates of any specific party. To make it easier to provide results, we will move \mbox{\hyperlink{classCandidate}{Candidate}} objects from the \char`\"{}candidates\char`\"{} vector into a vector of winners, \char`\"{}winners\char`\"{}. 

Implements \mbox{\hyperlink{classElectionSystem_a1e5cb35444beb775f688d6a4299c3629}{Election\+System}}.
